# Репозиторий по курсовой работе
## Студентки 3 курса ИВТ 1.2 Степановой Анны

## 📌 О чём работа

Разработка и развёртывание **веб-приложения для отслеживания курсов валют в реальном времени** с использованием **Docker** и автоматической настройкой **HTTPS** через Let's Encrypt.

---

## 🎯 Цель

Создать контейнеризированное веб-приложение, которое:
- Отображает актуальные курсы валют ЦБ РФ
- Работает в реальном времени через WebSocket
- Автоматически получает и обновляет SSL-сертификаты

---

## 🛠️ Что было сделано

### 1. Разработка приложения
- **Backend** — Python + Tornado (WebSocket, HTTP)
- **Frontend** — HTML, CSS, JavaScript
- **Источник данных** — API Центробанка РФ
- Реализован паттерн «Наблюдатель» для рассылки обновлений клиентам

### 2. Контейнеризация
- Созданы **Docker-образы** для всех компонентов
- Написан **docker-compose.yml** для оркестрации трёх сервисов

### 3. Автоматический HTTPS
- **nginx-proxy** — reverse-proxy и терминация SSL
- **acme-companion** — автоматическое получение и обновление сертификатов Let's Encrypt

### 4. Развёртывание
- Приложение запущено на **VPS-сервере (Ubuntu 22.04)**
- Настроен домен `stepanna.bizml.ru`
- Поддержка **WebSocket** через reverse-proxy

---

## 📊 Результат

✅ Приложение работает по адресу: **https://stepanna.bizml.ru**  
✅ Отображает курсы **USD, EUR, GBP, CNY, JPY, RUB**  
✅ Обновление каждые **30 секунд**  
✅ Защищённое соединение (**HTTPS**) с автоматическим продлением сертификата
